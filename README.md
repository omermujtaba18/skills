# skills-i-use-everyday

A collection of skills I use everyday.

## Install

Install every skill in this repo into your agent with the [skills.sh](https://www.skills.sh) CLI:

```bash
npx skills add omermujtaba18/skills
```

You'll be prompted to pick which skills to install. Skills are copied into your agent's skills
directory (e.g. `~/.claude/skills/` for user-wide, or `.claude/skills/` inside a project).

### Install a single skill

```bash
npx skills add omermujtaba18/skills/pr-review
```
